## 🛠 Using telbot on Windows

You can use telbot in three ways. Choose the one that suits you best.

### 1. Telegram Bot

- Open your Telegram app (desktop or mobile).
- Search for the telbot bot or follow instructions in the telbot application.
- Use the commands provided by telbot to check your Telkomsel account info.
- You don’t need to open the program; everything happens inside Telegram.

### 2. Terminal CLI (Command Line Interface)

- Open Command Prompt on your Windows PC (search for "cmd" in the Start menu).
- Type `telbot` and press Enter.
- Follow on-screen instructions to link your Telkomsel account and use commands.
- This way is good if you prefer typing commands or want quick access.

### 3. MCP Server (for AI Agents)

- This option is for more advanced users.
- The MCP Server lets AI tools manage your Telkomsel account through telbot.
- Setup details and configurations are available in the repository's documentation under the MCP Server section.
